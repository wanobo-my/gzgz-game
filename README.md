# GzGzのアイスつみ（GzGz Ice Cream Stacker）

アイスクリーム屋のキャラクター **GzGz（グズグズ）** が主役の、ブラウザで遊べるアイス積みゲームです。
降ってくるアイスをキャッチして高く積み上げましょう。テニスボール・太陽・タカに気をつけて！🍦

▶ **あそぶ**：https://wanobo-my.github.io/gzgz-stacker/

インストール不要。スマホ・PCのブラウザでそのまま遊べます。
